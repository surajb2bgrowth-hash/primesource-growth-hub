# PrimeSource IT Consulting - WordPress Theme

A modern one-page WordPress theme for PrimeSource IT Consulting.

## Installation

1. Create a folder called `primesource` in your computer
2. Copy all these files into that folder
3. Create a 1200x900px screenshot.png for theme preview
4. Zip the `primesource` folder
5. Go to WordPress Admin → Appearance → Themes → Add New → Upload Theme
6. Upload the zip file and activate

## Customization

Go to **Appearance → Customize** to:
- Upload your logo
- Edit hero section text
- Update contact information
- Add social media links

## Custom Post Types

- **Services**: Add your services from the WordPress admin
- **Case Studies**: Showcase your projects
- **Testimonials**: Add client testimonials

## Creating a Menu

1. Go to Appearance → Menus
2. Create a new menu called "Primary Menu"
3. Add custom links with anchor URLs:
   - Home → #hero
   - Services → #services
   - About → #about
   - Case Studies → #case-studies
   - Blog → #blog
   - Contact → #contact
4. Assign to "Primary Menu" location

## Adding Images

Create an `assets/images/` folder and add:
- about-team.jpg (600x400)
- case-study-1.jpg, case-study-2.jpg, case-study-3.jpg (600x400)
- blog-placeholder.jpg (600x400)
- placeholder.jpg (600x400)
